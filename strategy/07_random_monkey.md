## 🎯 매수 조건

```python
import random

def generate_signal(self, df):
    rand = random.random()
    
    if rand < 0.1:  # 10% 확률
        return 'buy'
    elif rand < 0.2:  # 10% 확률
        return 'sell'
    else:
        return 'hold'
```

**그냥 랜덤!**
- 10% 확률로 매수
- 10% 확률로 매도
- 80% 확률로 관망
