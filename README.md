# 🤖 업비트 자동매매 봇

업비트 API를 이용한 자동매매 시스템

## 📋 기능

- [ ] 실시간 시세 모니터링
- [ ] 기술적 지표 기반 매매 (RSI, MACD, 볼린저 밴드)
- [ ] 백테스팅 시스템
- [ ] 손절/익절 자동화
- [ ] 텔레그램 알림
- [ ] 포트폴리오 관리

## 🛠 설치

```bash
pip install -r requirements.txt
```

## ⚙️ 설정

1. `.env.example`을 `.env`로 복사
2. 업비트 API 키 입력
3. 매매 전략 설정

## 🚀 실행

### 백테스팅
```bash
python backtest.py
```

### 실전 매매
```bash
python bot.py
```
